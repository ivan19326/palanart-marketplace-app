# PALAN HYPE Marketplace

Маркетплейс исполнителей и услуг для мероприятий.

## Публикация

Проект настроен на деплой через GitHub Pages из папки `marketplace_build`.

После загрузки репозитория на GitHub:

1. Откройте `Settings -> Pages`.
2. Убедитесь, что источник публикации работает через GitHub Actions.
3. Пушьте изменения в ветку `main`.

## Установка на телефон

После публикации откройте `index.html` на сайте:

- на Android в Chrome: `Меню -> Установить приложение`
- на iPhone в Safari: `Поделиться -> На экран "Домой"`

PWA использует:

- `manifest.json`
- `service-worker.js`
- standalone-режим
- иконки для установки
